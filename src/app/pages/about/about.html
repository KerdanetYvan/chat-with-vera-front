À propos de ChatWithVera

Le projet

ChatWithVera est une application web développée dans le cadre d'un projet pédagogique. Cette plateforme facilite l'accès au service VERA, un outil de vérification des faits basé sur l'IA.

Technologies

  • Frontend : Angular 21
  • Backend : NestJS
  • Base de données : MongoDB Atlas (Europe/Paris)
  • IA : Google Gemini + VERA
  • Hébergement : Vercel

Protection des données (RGPD)

  • Minimisation des données collectées
  • Non-stockage des messages du chatbot
  • Chiffrement HTTPS et tokens sécurisés
  • Protection spécifique pour les mineurs
